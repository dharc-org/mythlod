Technical Documentation

h

The webiste template has been designed by Start Bootstrap. Icons made by Flaticon (e.g. loading spinner).

knowledge organisation

Figures representing the conceptual model and the case study has been drown with Graffoo. Additionaly, the case study has been also represented with Brick-Studio visualisation tool.

catalogue

The catalogue structure has been developed on the top of two open-source javascript libraries:

  • eikes faceted search, which provides the faceted search function.
  • The sidebar provides the possibility to close the sidebar (after faceted search or if no search needed) and browse manually the catalogue in screen full-width. The two libraries has been integrated to create a user-friendly enviroment despite the huge amount of browsanble entities involved in faceted search.
